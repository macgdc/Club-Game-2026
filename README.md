# MacGDC Club Game 2026
This is an open source game the McMaster University Game Development Club is working on. The goal is to finish the game before the end of the 2026 winter semester (April 30th) and if possible submit it to the Level Up Showcase (April 7th).

The idea: 3D Dead Cells like game.

## Setup
Ensure [Git LFS](https://git-lfs.com/) is installed and setup.
Using Godot version 4.6 stable.
